indexOf()를 이용해서 CSS 속성 검색하기

CSS 속성 검색하기

  • ALL : all 속성은 CSS 속성을 재설정하는데 사용할 수있는 약식 속성입니다.
  • Animation : Animation 속성은 애니메이션 속성을 설정하기 위한 약식 속성입니다.
  • Animation-delay : Animation-delay 속성은 애니메이션이 시작되는 시간을 설정하기 위한 약식 속성입니다.
  • Animation-direction : Animation-direction 속성은 애니메이션이 움직이는 방향을 설정하기 위한 약식 속성입니다.
  • Animation-fill-mode : Animation-fill-mode 속성은 애니메이션이 끝난 후의 상태를 설정하기 위한 약식 속성입니다.
  • Animation-iteration-conut : Animation-iteration-conut 속성은 애니메이션이 반복 횟수를 설정하기 위한 약식 속성입니다.
  • Animation-name : Animation-name 속성은 애니메이션 키프레임 이름을 설정하기 위한 약식 속성입니다.
  • Animation-play-state : Animation-play-state 속성은 애니메이션의 진행상태를 설정하기 위한 약식 속성입니다.
  • backdrop-filter : backdrop-filter 속성은 요소 뒤에 필터효과를 설정합니다.
  • backdrop-bisibility : backdrop-bisibility 속성은 요소 뒷면 출력 여부를 설정합니다.
  • caption-side : caption-side 속성은 표(table)의 캡션의 위치를 설정합니다.

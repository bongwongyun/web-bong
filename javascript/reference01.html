자바스크립트 선언문

선언문은 자바스크립트 코드를 작성할 영역을 선언하는 것이라고 이해하면 됩니다. 주로 <head><body> 태그영역에 사용되지만 보통은 <head> 영역에 많이 태그합니다. <script> 라는 태그로 선언문이 시작된곳부터 </script> 라는 태그로 종료된 곳 까지 스크립트 영역 이라고 부릅니다. 
<!DOCTYPE html>
                <html lang="ko">
                    <head>
                        <meta charset="UTF-8">
                        <title>document</title>
                        <script>
                        document.write("환영합니다");
                        </script>  
                    <head>
                    ......
                </html>

코드 입력시 주의사항

1.대 소문자를 구분해야 합니다.

2.한줄 작성 후 세미콜론을 찍어주셔야합니다.

document.write("환영합니다") document.write("안녕하세요")
                document.write("환영합니다"); document.write("안녕하세요");

3. 가독성을 위해 한줄만 작성하시는게 좋습니다.

4. 문자형 데이터를 작성할 때 작은 따옴표('), 큰 따옴표(") 의 겹치지 않게 오류를 주의해야 합니다.

 잘못된 예시 : document.write("환영합니다" 자바스크립트 겹침 오류 입니다"라고 알려드립니다")
 옳은 예시 : document.write('환영합니다" 자바스크립트 겹침 오류 입니다"라고 알려드립니다');

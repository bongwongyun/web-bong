HTML


HTML은 HyperText Markup Language의 약자로 웹 페이지를 구성하는 마크업 언어입니다.

자바스크립트와 CSS 등을 같이 사용하여 본문과 외관의 배치를 정의할 수 있습니다.


<head>
<title>제목</title>
</head>

태그(tag)


HTML 태그는 태그 이름을 꺾쇠 괄호(<>)로 감싸서 표현합니다.

HTML 태그는 보통 시작 태그(start tag, opening tag)와 종료 태그(end tag, closing tag)의 한 쌍으로 구성됩니다. 종료 태그는 시작 태그와 전부 똑같지만, 태그 이름 앞에 슬래시(/)가 존재합니다. 태그에 따라 시작 태그만 있고 종료 태그가 없는 태그도 존재합니다.


<태그>
</태그>

<!DOCTYPE html>


현재 문서가 HTML5 문서임을 명시합니다.


<!DOCTYPE html>

<html>


HTML 문서의 루트(root)요소를 정의합니다.


<html>
</html>

<head>


HTML 문서의 메타데이터(metadata)를 정의합니다.

- 메타데이터(metadata)란 HTML 문서에 대한 정보(data)로 웹 브라우저에는 직접적으로 표현되지 않는 정보를 의미합니다.

- 이러한 메타데이터는 <title>,<style>,<meta>,<link>,<script>,<base>태그 등을 이용하여 표현할 수 있습니다.


<head>
</head>

<title>


HTML 문서의 제목을 정의하며, 다음과 같은 용도로 사용됩니다.

- 웹 브라우저의 툴바(toolbar)에 표시됩니다.

- 웹 브라우저의 즐겨찾기(favorites)에 추가할 때 즐겨찾기의 제목이 됩니다.

- 검색 엔진의 결과 페이지에 제목으로 표시됩니다.


<title>
</title>

<body>


웹 브라우저를 통해 보이는 내용(content) 부분입니다.


<body>
</body>

<h1~h6>


제목(heading)을 나타냅니다.


<h1>
</h1>

<p>


단락(paragraph)을 나타냅니다.


<p>
</p>

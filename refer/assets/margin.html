CSS margin

margin

margin 속성은 요소의 여백을 설정합니다.

정의

  • margin-top : 여백의 위쪽을 설정합니다.
  • margin-right : 여백의 바깥 오른쪽을 설정합니다.
  • margin-boottom : 여백의 아래쪽을 설정합니다.
  • margin-lfte : 여백의 바깥 왼쪽을 설정합니다.

margin: 10px; /* 위쪽 오른쪽 아래쪽 왼쪽:10px */

margin: 10px 11px; /* 위쪽 아래쪽:10px 왼쪽 오른쪽 11:px */

margin: 10px 11px 12px; /* 위쪽:10px 오른쪽 왼쪽:11px 아래쪽:10px */

margin: 10px 11px 12px 13px; /* 위쪽:10px 오른쪽:11px 아래쪽:12px 왼쪽:13px */

margin: auto; /* 자동값,기본값 */

margin: 0 auto; /* 가운데 정렬 (블록 구조) */ /* 패딩도 같다*/

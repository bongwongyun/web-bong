HTML 레퍼런스 북

HTML 레퍼런스 속성과 설명을 나타내는 표입니다.
속성 설명
<a> <a> 태그는 다른 페이지 이동을 설정합니다.
<abbr> <abbr> 태그는 준말 또는 머리 글자를 나타냅니다.
<acronym> <acronym> 태그는 <HTML> 요소의 목록을 제공합니다.
<address> <address> 태그는 <HTML> 요소의 사람, 단체, 등에 대한 정보를 나타냅니다.
<applet> <applet> 태그는 문서에 포함되는 애플릿(웹 페이지에 포함되어 작은 기증을 수행하는 프로그램)을 정의할 때 사용합니다.
<area> <area> 태그는 이미지 맵에서 하이퍼링크가 연결될 영역을 정의할 때 사용합니다.
<article> <article> 태그는 해당 문서나 페이지 또는 사이트와는 완전히 독립적으로 구성할 수 있는 요소를 정의할 때 사용합니다.
<aside> <aside> 태그는 페이지의 다른 콘텐츠들과 약간의 연관성을 가지고 있지만, 해당 콘텐츠들로부터 분리시킬 수 있는 콘텐츠로 구성된 페이지 영역을 정의할 때 사용합니다.
<audio> <audio> 태그는 오디오을 넣는 태그입니다.
<b> <b> 태그는 묶은 내용들을 굵게 표시합니다.
<base> <base> 태그는 해당 문서의 모든 상대 주소 (relatice URL)에 대한 기본 URL(base URL)과 target 속성값을 정의할 때 사용합니다.
<basefont> <basefont> 태그는 해당 문서의 모든 텍스트를 위한 기본 색상, 크기, 글꼴 집합(font-family) 등을 정의할 때 사용합니다.
<bdi> <bdi> 태그의 BDI는 BI-Directional lsolation을 의미하며, 주위의 텍스트들과는다른 방향으로 써지는 텍스트 영역을 정의할 때 사용합니다.
<bdo> <bdo> 태그의 BDO는 BI-Directional Override를 의미하며, 자식 요소의 텍스트 방향성(text directionality)을 정의할 때 사용합니다.
<big> <big> 태그는 글자 크기가 큰 텍스트를 정의할 때 사용합니다.
<blockquote> <blockquote> 태그는 다른 출처로부터 인용된 블록을 정의할 때 사용하며, 이렇게 인용된 블록을 브라우저는 보통 들여쓰기를 사용하여 표현합니다.
<blink> <blink> 태그는 텍스트를 깜박이게 설정합니다.
<body> <body> 태그는 해당 HTML문서의 텍스트, 하이퍼링크, 이미지, 리스트 등과 같은 모든 콘텐츠를 포함하는 영역을 정의할 때 사용합니다.
<br> <br> 태그는 텍스트 내의 줄바꿈(line-break)을 정의할 때 사용합니다.
<button> <button> 태그는 클릭할 수 있는 버튼을 정의할 때 사용합니다.
<canvas> <canvas> 태그는 주로 자바스크립트와 같은 스크립트를 이용하여 그래픽 콘텐츠를 그릴 때 사용합니다.
<caption> <caption> 태그는 table 요소의 제목을 설정합니다.
<center> <center> 태그는 가운데로 정렬되는 텍스트를 정의할 때 사용합니다.
<cite> <cite> 태그는 책이나 음악, 영화, 예술 작품 등과 같은 창작물의 제목을 정의할 때 사용합니다.
<code> <code> 태그는 코드를 나타내는 텍스트를 설정합니다.
<col> <col> 태그는 <colgroip> 요소에 속하는 갈 열의 속성을 설정합니다.
<colgroup> <colgroup> 태그는 표의 열을 묶는 그룹을 설정합니다.
<data> <data> 태그는 특정 콘텐츠에 기계가 읽을 수 있는(machine- readabble) 형태의 값(value)을 덧붙여 나타낼 때 사용합니다.
<datalist> <datalist> 태그는 <input> 요소에서 사용하기 위한 옵션들의 리스트를 미리 정의할 때 사용합니다.
<dd> <dd> 태그는 용어와 그에 대한 설명을 리스트 형식으로 보여주는 <dl> 요소에서 설명 (description) 부분을 정의할 때 사용합니다.
<del> <del> 태그는 텍스트 한가운데 라인을 추가하여 문서에서 삭제된 텍스트를 표현할 때 사용합니다.
<details> <details> 태그는 사용자가 직접 조작하여 보거나 숨길 수 있는 부가적인 세부사항(additional details)을 명시할 때 사용합니다.
<dfn> <dfn> 태그는 HTML는 용어(term)의 정의를 나타낼 때 사용합니다.
<dialog> <dialog> 태그는 대화 상자(dialog box)나 대화 윈도우(dialog window)를 정의할 때 사용합니다.
<dir> <dir> 태그는 HTML 리스트를 정의할 때 사용합니다.
<div> <div> 태그는 문서의 영역을 나누거나 그룹화 할 때 설정합니다.
<dl> <dl> 태그는 태그는 용어(term)그에 대한 설명(description)을 리스트 형식으로 정의할 때 사용합니다.
<dt> <dt> 태그는 용어와 그에 대한 설명을 리스트 형식으로 보여주는 <di>요소와 해당 용어에 대한 설명을 나타내는 <dd>요소로 구성됩니다.
<em> <em> 태그는 텍스를 강조할 때 설정합니다.
<embed> <embed> 태그는 외부 애플리케이션이나 대화형 콘텐츠를 포함시킬 수 있는 컨테이너(container)를 정의할 때 사용합니다.
<fieldset> <fieldset> 태그는 <formt> 요소에서 연관된 요소들을 하나의 그룹으로 묶을 때 사용합니다.
<figcaption> <figcaption> 태그는 이미지나 다이어그램, 사진과 같이 문서의 주요 흐름과는 독립적인 콘텐츠를 정의하는 <figure>요소의 캡션(caption)을 정의할 때 사용합니다.
<figure> <figure>태그는 삽화나 다이어그램, 사진 등과 같이 문서의 주요 흐름과는 독립적인 콘텐츠를 정의할 때 사용합니다.
<font> <font> 태그는 텍스트의 폰트(font), 색상 그리고 크기를 정의할 때 사용합니다.
<footer> <footer> 태그는 문서나 특정 섹션(section)의 푸터(footer)를 정의할 때 사용합니다.
<form> <form> 태그는 사용자로부터 입력을 받을 수 있는 HTLM 입력 폼(form)을 정의할 때 사용합니다.
<frame> <frame> 태그는 <frameset> 요소에 포함되는 하나의 프레임(frame)을 정의할 때 사용합니다.
<frameset> <frameset> 태그는 문서의 레이아웃을 구성하기 위해 사용되는 프레임(frame)들의 집합을 정의할 때 사용합니다.
<h1> ~ <h6> <h1> ~ <h6> 태그는 섹션의 제목을 설정합니다.
<head> <head> 태그는 제목과 스크립트, 스타일 시트, 메타 정보 등을 제공하는 영역입니다.
<header> <header> 태그는 문서나 특정 섹션(section)의 헤더(header)를 정의할 때 사용합니다.
<hgroup> <hgroup> 태그는 HTML 문서 또는 섹션의 헤더를 정의하는 데 사용됩니다.
<hr> <hr> 태그는 요소간의 주제별 구분을 설정합니다.
<html> <html> 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
<i> <i> 태그는 글자를 기울게 만들어줍니다.
<iframe> <iframe> 태그는 인라인 프레임(inline frame)을 정의할 때 사용합니다
<img> <img> 태그는 웹 문서에 이미지를 설정합니다.
<input> <input> 태그는 사용자로부터 입력을 받을 수 있는 입력 필드(input filed)를 정의할 때 사용합니다.
<ins> <ins> 태그는 텍스트 아래쪽에 라인(line)을 추가하여 문서에 추가된 텍스트를 표현할 때 사용합니다.
<kbd> <kbd> 태그는 키보드(keyboard) 입력을 나타낼 때 사용합니다.
<keygen> <keygen> 태그내의<input> 입력값을 쌍기방식(개인키는 사용자컴에, 공공키는 서버에 전달)으로 암호화
<label> <label> 태그는 사용자 인터페이스(UI) 요소의 라벨(label)을 정의할 때 사용합니다.
<legend> <legend> 태그는 <fieldset>요소의 캡션(caption)을 정의할 때 사용합니다.
<li> <li> 태그는 목록의 항목을 설정합니다.
<link> <link> 태그는 외부 파일을 연결할 때 설정합니다.
<main> <main> 태그는 웹 문서의 역할을 동일하게 설정합니다.
<map> <map> 태그는 클라이언트 사이드 이미지맵(client-side image-map)을 정의할 때 사용합니다.
<mark> <mark> 태그는 형광펜으로 칠한 것처럼 하이라이트된(highlighted) 텍스트를 정의할 때 사용합니다.
<marquee> <marquee> 태그는 텍스트를 좌우상하로 움직이는 모션을 취할 때 사용합니다.
<menu> <menu> 태그는 목록과 메뉴를 정의합니다
<menuitem> <menuitem> 태그는 마우스 오른쪽을 눌렀을 때 나타나는 팝업 페이지에 명령이나 메뉴 사항을 추가합니다.
<meta> <meta> 태그는 해당 문서에 대한 정보인 메타데이터(metadata)를 정의할 때 사용합니다.
<meter> <meter> 태그는 분수 값이나 게이지(gauge) 내에서 특정 스칼라 값(scalar measurement)이 어느 정도인지를 표현할 때 사용합니다.
<nav> <nav> 태그는 다른 페이지 또는 현재 페이지의 다른 부분과 연결되는 네비게이션 링크(navigation links)들의 집합을 정의할 때 사용합니다.
<noframes> <noframes> 태그는 프레임(frame) 기능을 사용하지 않도록 설정했거나, 프레임 기능을 지원하지 않는 브라우저를 위한 별도의 콘텐츠를 정의할 때 사용합니다.
<noscript> <noscript> 태그는 클라이언트 사이드 스크립트(client-side scripts)를 사용하지 않도록 설정했거나, 스크립트를 지원하지 않는 브라우저를 위한 별도의 콘텐츠를 정의할 때 사용합니다.
<object> <object> 태그는 외부 리소스의 종류에 따라 문서 내에서 처리하는 방식을 변경할 수 있는 임베디드 객체(embedded object)를 정의할 때 사용합니다.
<ol> <ol> 번호가 붙는 리스트 태그입니다. 자동 줄 바뀜이 됩니다.
<optgroup> <optgroup> 태그는 옵션 메뉴를 제공하는 드롭다운 리스트(drop-down list)에서 사용되는 옵션들의 그룹을 정의할 때 사용합니다.
<option> <option> 태그는 옵션 메뉴를 제공하는 드롭다운 리스트(drop-down list)에서 사용되는 하나의 옵션을 정의할 때 사용합니다.
<output> <output> 태그는 스크립트 등에 의해 수행된 계산의 결과나 사용자의 액션에 의한 결과를 나타낼 때 사용합니다.
<p> <p> 태그는 단락을 나타낼 때 설정합니다.
<param> <param> 태그는 <Object> 요소에 의해 호출되는 플러그인의 매개변수(parameter)를 정의할 때 사용합니다.
<picture> <picture> 태그는 <img>요소의 다중 이미지 리소스(multiple image resources)를 위한 컨테이너를 정의할 때 사용합니다.
<pre> <pre> 태그는 HTML에 작성한 내용을 화면에 그대로 표현합니다.
<pregress> <pregress> 태그는 작업의 진행률을 나타낼 때 사용합니다.
<q> <q> 태그는 짧은 인용구(short quotation)를 정의할 때 사용합니다.
<rp> <rp> 태그는 루비 주석(ruby annotation)을 지원하지 않는 브라우저에서만 표시될 문자를 정의할 때 사용합니다.
<rt> <rt> 태그는 한자의 음과 뜻을 설명하는 태그입니다.
<rtc> <rtc> 태그는 루비 문자의 텍스트 영역을 정의합니다.
<ruby> <ruby> 태그는 한자를 표시 해줍니다.
<s> <s> 태그는 텍스트 한가운데 라인(line)을 추가하여 더 이상 정확하지 않거나 관련이 없는 텍스트를 표현할 때 사용합니다.
<samp> <samp> 태그는 컴퓨터 프로그램의 샘플 또는 인용 출력(output)을 나타낼 때 사용합니다.
<script> <script> 태그는 자바스크립트와 같은 클라이언트 사이드 스크립트(client-side scripts)를 정의할 때 사용합니다.
<section> <section> 태그는 HTML 문서에 포함된 독립적인 섹션(section)을 정의할 때 사용합니다.
<select> <select> 태그는 옵션 메뉴를 제공하는 드롭다운 리스트(drop-down list)를 정의할 때 사용합니다.
<small> <small> 태그는 글자 크기가 작은 텍스트를 정의할 때 사용합니다.
<source> <source> 태그는 <audio> 요소나 <video>, <picture> 요소에서 사용할 수 있는 다중 미디어 자원(multiple media resources)을 정의할 때 사용합니다.
<span> <span> 태그는 HTML 문서에서 인라인 요소(inline-element)들을 하나로 묶을 때 사용합니다.
<strike> <strike> 태그는 텍스트 한가운데 라인을 추가할 때 사용합니다.
<strong> <strong> 태그는 텍스트의 중요성을 강조할 때 설정합니다.
<style> <style> 태그는 웹 문서에 스타일 정보를 설정합니다.
<sub> <sub> 태그는 아랫첨자(subscript) 텍스트를 표현할 때 사용합니다.
<summary> <summary> 태그는 <details> 요소에 의해 생성되는 대화형 위젯에서 기본적으로 보이는 제목을 정의할 때 사용합니다.
<sup> <sup> 태그는 윗첨자(superscript) 텍스트를 표현할 때 사용합니다.
<SVG> <SVG> 웹상에서 2차원 벡터 그래픽을 표현하는 XML 기반의 마크업 언어.
<table> <table> 태그는 표를 설정합니다.
<tbody> <tbody> 태그는 표의 본문 영역을 그룹화 합니다.
<td> <td> 태그는 표와 열(셀)을 설정합니다.
<template> <template> 태그는 추가되거나 복사될 수 있는 HTML 요소들을 정의할 때 사용합니다.
<textarea> <textarea> 태그는 사용자가 여러 줄의 텍스트를 입력할 수 있는 텍스트 입력 영역을 정의할 때 사용합니다.
<tfoot> <tfoot> 태그는 표의 푸터 영역을 그룹화합니다.
<th> <th> 태그는 행이나 열에 타이틀을 설정합니다.
<thead> <thead> 태그는 표의 헤더 영역을 그룹화합니다.
<time> <time> 태그는 사람이 읽을 수 있는(human-readable) 형태의 날짜와 시간 데이터를 정의할 때 사용합니다.
<title> <title> 태그는 브라우저 제목 또는 탭의 표시되는 문서의 제목을 설정합니다.
<tt> <tt> 태그는 텔레타이프 텍스트(teletype text)를 나타낼 때 사용합니다.
